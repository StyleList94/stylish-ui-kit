# Stylish UI Kit

Stylish UI kit for React applications

## Features

- Storybook
- Testing(Jest, React Testing Library)

## Get Started

### Run Storybook

```bash
yarn storybook
```

### Build

#### Package

```bash
yarn build
```

#### Storybook

```bash
yarn build-storybook
```

### Testing

```bash
# Dev(watching)
yarn test

# CI
yarn test:ci
```
